Simple, advertising-free Android phone app (Tiramisu & up) for remote control. It works in conjunction with a cellular 4G/LTE relay box, available fairly cheaply on eBay, Amazon, etc. It lets you create an easily accessible screen of buttons, each of which will perform whatever action you assign. It needs code contributions!
